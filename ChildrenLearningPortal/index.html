Welcome!

This is a collection of interactive games designed for children, each game aims to further develop the childs skill set in a variety of tasks that involve spelling, writing and simple math. It is designed in a fun way so that your kid will never become bored! There are different levels of difficulty for different ages. To select and play a level click on the Choose level tab above this one. You can see each game's rules by clicking on the rules tab that is located on the right corner on each level. The instructions are written in a way that every kid can understand in case that he get's stuck at some point.

Question Mark Button

If you need help about something you can get additional information by clicking on the question mark button, you can find it at any game or page in the top right corner
